# Personal Portfolio Website

A simple personal portfolio website built using HTML and CSS. The website consists of two pages: a Home page that introduces me and my work, and a Contact page where visitors can get in touch with me.

## Features

- Clean and responsive design
- Home page with personal introduction
- Contact page for communication details
- Built using pure HTML and CSS
- Beginner-friendly project structure

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
portfolio-website/

```

## Purpose

This project was created to practice web development fundamentals and build a basic portfolio website to showcase my online presence.
